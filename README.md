# HealthScope

**HealthScope** is an advanced machine learning platform that provides real-time health risk assessments for three chronic conditions: Diabetes, Heart Disease and Obesity. The system combines state-of-the-art ML models with a user-friendly interface to provide personalized health insights.

---

## Built With

- **Frontend:** Next.js, Tailwind CSS

- **Backend:** Python, FastAPI, Uvicorn, Pydantic

- **Data Science:** Pandas, NumPy, SciPy, Matplotlib, Seaborn, Plotly

- **Machine Learning:** Scikit-learn, XGBoost, Random Forest, SHAP
